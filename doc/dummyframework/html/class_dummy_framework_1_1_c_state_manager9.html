Detailed Description

State manager for effects.

The state manager object helps to filter out redundant renderstate and texture state switches. However this requires that you use it for these kind of calls.

If you use D3DXCreateEffectXX(), then you have to set the state manager for it manually, and you have to use the _DUMMY_EFFECT_DONT_SAVE_ flag when calling ID3DXEffect::Begin().
